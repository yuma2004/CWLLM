# 懸念点・問題点メモ

## 実装/運用面

- `prisma generate` が Windows 環境で `EPERM` になることがある（`query_engine-windows.dll.node` のロック）
  - 対応: Node/エディタ/ウイルス対策のロック解除後に再実行、または `migrate:dev -- --skip-generate` で暫定回避
- Chatwork トークン未設定のまま同期を実行すると 400 で失敗する
  - 対応: `.env` の `CHATWORK_API_TOKEN` 設定が必要

## テスト/CI

- React Router の将来フラグ警告がテスト実行時に出る
  - 対応: `future` フラグで opt-in すれば警告は抑制可能

## 仕様差分

- なし
