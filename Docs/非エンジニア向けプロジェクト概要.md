# 非エンジニア向けプロジェクト概要

## これは何をするシステムか

CWLLMは、Chatwork上の会話を中心に、顧客情報・案件・タスクをひとつにまとめて管理するための業務システムです。  
「誰が、どの顧客に、いま何を対応中か」をチームで見える化し、対応漏れを減らします。

## 解決したい課題

- チャットと顧客管理が分かれていて、経緯が追いづらい
- 担当者ごとのタスク状況が見えにくい
- 案件の進捗と会話履歴がつながっていない
- 引き継ぎ時に情報が抜けやすい

## 主な利用者

- 営業担当
- 営業マネージャー
- カスタマーサポート担当
- 管理者（ユーザー管理・運用管理）

## できること

### 1. 会話情報を自動で取り込む

- Chatworkのルーム/メッセージを同期
- 会話履歴を企業・案件に紐づけて蓄積

### 2. 顧客と案件の情報を一元管理する

- 企業情報、担当者情報を管理
- 企業ごとのプロジェクト（案件）と卸情報を管理

### 3. タスクを運用しやすくする

- タスクを一覧とカンバンで確認
- 担当者、期限、ステータスで管理
- 進捗の抜け漏れを早期に発見

### 4. 会話内容の要点整理を支援する

- AIで要約ドラフトを作成
- 長い会話の要点把握を短時間で実施

### 5. 全体状況を把握しやすくする

- ダッシュボードで全体感を可視化
- 検索機能で必要情報に素早く到達
- フィードバックで運用改善を継続

## 導入効果（期待値）

- 情報探索の時間短縮
- タスク漏れ・対応漏れの削減
- 顧客対応のスピード向上
- 引き継ぎ品質の向上
- マネージャーの状況把握コスト削減

## 日常利用のイメージ

1. 朝にダッシュボードで未対応/期限近いタスクを確認
2. 企業詳細画面で関連会話と案件状況を確認
3. 必要な対応をタスク化し、担当者と期限を設定
4. 必要に応じてAI要約を使って会話の要点を共有
5. 終業時に進捗を更新し、翌日の対応を明確化

## このシステムの位置づけ

CWLLMは、チャット履歴を中心に営業活動の実務情報をつなげる「業務運用のハブ」です。  
単なる顧客台帳ではなく、日々の会話と実行タスクを結びつけることで、現場で使える運用基盤を提供します。
